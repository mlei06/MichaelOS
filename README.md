# MichaelOS

My personal site, styled as a desktop OS. Live at [michael-lei.com](https://michael-lei.com).

Built with [Astro](https://astro.build) + React. Static output, deployed to S3 + CloudFront.

## Local dev

```bash
npm install
npm run dev      # http://localhost:4321
npm run build    # static output to dist/
npm run preview  # serve the build locally
```
